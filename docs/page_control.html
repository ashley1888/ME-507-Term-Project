Controller Task

This task is the controller task of the claw machine. It has the user enter a value between 1 and 64, where each value represents a position on an 8x8 chessboard. Once a value has been entered, the controller task sends the responding x and y-values to the x and y-motors. The purpose of the value is so that the motor encoders know when they have reached their destination. It's source code can be found at:
