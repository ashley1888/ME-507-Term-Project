Detailed Description

This file is the controller task of the claw machine. It has the user enter a value between 1 and 64, where each value represents a position on an 8x8 chessboard. Once a value has been entered, the controller task sends the responding x and y-values to the x and y-motors. The purpose of the value is so that the motor encoders know when they have reached their destination.

void task_controller ( void *  p_params)

Task which reads the user's input sends the appropriate values through shared variables.

Assigns an x and y-position to every square of an 8x8 chessboard and sends the appropriate value depending on the user's input. When the other tasks are running, this task will wait until they send back a "done" indicator, forcing the user to wait until then.
